■ 개인 과제
오늘 개인 과제 기능 구현에 집중했다. 
- 각 요일 별 랜덤 정수로 구현된 평균 구하기 
- reset 버튼 누르면 평균 0으로 상태 변화하기 
- 삼각형 버튼 눌러서 요일별 페이지 이동
- 선택한 요일 페이지 보여주기 
- 평점 남기기 버튼 누르면 이전 페이지 이동

위 목록이 오늘 구현한 기능이다. 
이제 남은 기능 구현은 동그라미 눌러서 평점 입력하기!! 
이 기능 구현이 생각보다 오래걸릴 것 같아서 걱정이지만... 
그래도 이만큼 해낸건 정말 다행이라고 생각한다. 

주특기 입문 마지막날 하루도 잘 이겨내보자!
